# Agentic Career Counseling Companion — Backend

## Quick Start

```bash
cd backend
python -m venv venv
venv\Scripts\activate          # Windows
# source venv/bin/activate     # macOS / Linux
pip install -r requirements.txt
copy .env.example .env         # Windows
# cp .env.example .env         # macOS / Linux
# Edit .env and fill in your IBM credentials
python app.py
```

The API will be available at http://localhost:5000

## Verify Installation

Visit http://localhost:5000/api/health — should return HTTP 200 with
Granite model checks passing.

## API Endpoints

| Method | Path | Description |
|--------|------|-------------|
| GET | /api/health | Verify Granite models and knowledge base |
| POST | /api/validate | Validation Agent — extract + gap-detect profile |
| POST | /api/profile | Profile Agent — summary, score, tier |
| POST | /api/recommend | Career Recommendation Agent — top 3 careers |
| POST | /api/skillgap | Skill Gap Agent — skills and tools to learn |
| POST | /api/roadmap | Roadmap Agent — 30/60/90 day plan |
| POST | /api/pipeline | Full pipeline — all 5 agents in one request |
| POST | /api/stt | Watson STT fallback — transcribe audio |
| DELETE | /api/session | Clear all session data (Start Over) |

## Pipeline Order

Endpoints must be called in this sequence per session:

```
/api/validate → /api/profile → /api/recommend → /api/skillgap → /api/roadmap
```

Or call `/api/pipeline` for all five in one request.

## IBM Cloud Foundry Deployment

```bash
ibmcloud login
ibmcloud target --cf
cd backend
ibmcloud cf push
ibmcloud cf set-env career-counseling-companion IBM_API_KEY <your_key>
ibmcloud cf set-env career-counseling-companion IBM_PROJECT_ID <your_project_id>
ibmcloud cf set-env career-counseling-companion IBM_WATSONX_URL https://us-south.ml.cloud.ibm.com
ibmcloud cf set-env career-counseling-companion FLASK_SECRET_KEY <random_secret>
ibmcloud cf restage career-counseling-companion
```

## Environment Variables

| Variable | Required | Description |
|---|---|---|
| IBM_API_KEY | ✅ Required | IBM Cloud IAM API key |
| IBM_PROJECT_ID | ✅ Required | watsonx.ai project identifier |
| IBM_WATSONX_URL | ✅ Required | Regional watsonx.ai endpoint |
| FLASK_SECRET_KEY | ✅ Required | Flask session encryption key |
| WATSON_STT_API_KEY | Optional | Watson STT API key (STT fallback) |
| WATSON_STT_URL | Optional | Watson STT service URL |
| FLASK_ENV | Optional | development (default) or production |
| LOG_LEVEL | Optional | DEBUG / INFO / WARNING / ERROR (default: INFO) |
| SESSION_LIFETIME_SECONDS | Optional | Session TTL in seconds (default: 3600) |
